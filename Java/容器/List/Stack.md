# Stack

![][image-1]

Stack继承自Vector，通过五个操作对Vector进行拓展：
1. `empty()`：测试堆栈是否为空。
2. `peek()`查看顶部对象，但并不移除。
3. `pop()`：移除顶部对象并返回。
4. `push(E item)`：把项压入栈顶部。
5. `search(Object o)`：返回对象在堆栈中的位置，以1为基数。
